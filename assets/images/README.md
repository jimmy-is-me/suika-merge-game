# 🖼️ 圖片素材放這裡

請把你的圖片命名為以下格式，放在此資料夾（`assets/images/`）：

| 檔案名稱 | 對應等級 | 建議大小 |
|---|---|---|
| `item-1.png` | 等級 1（最小） | 200×200px+ |
| `item-2.png` | 等級 2 | 200×200px+ |
| `item-3.png` | 等級 3 | 200×200px+ |
| `item-4.png` | 等級 4 | 200×200px+ |
| `item-5.png` | 等級 5（最大） | 200×200px+ |

## 圖片規格

- **格式**：PNG（建議帶透明背景）
- **尺寸**：正方形，最小 200×200px，建議 512×512px
- **形狀**：圖片主體盡量接近圓形，外圍保留透明
- **命名**：必須是 `item-1.png` 到 `item-5.png`

## 注意事項

如果此資料夾沒有圖片（或圖片路徑錯誤），遊戲會自動使用內建的色塊 fallback（A/B/C/D/E），不會出錯。

你也可以在 `index.html` 的 `ITEMS` 陣列裡改成其他路徑或圖片名稱。
